

# Restaurant Order Management System

This project is a web-based restaurant order management system.

Customers can browse menu items and place orders through the website.
The admin panel allows staff to view and manage incoming orders.

## Technologies Used

HTML  
CSS  
JavaScript  
Node.js  

## Installation

1. Clone the repository
git clone https://github.com/sermatudu/Restaurant-order-management-system

2. Install dependencies
npm install

3. Start the server
node SERVER/server.js

## Project Structure

CLIENT  - Frontend website files  
SERVER  - Backend server code
